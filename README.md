# Vircade
## Set up
`$ cd Vircade`

## To install package
`$ flutter pub get`

## To run flutter
`$ flutter run`


# Cloud function
## Set up
```
npm install -g firebase-tools
firebase init
```

## Deploy 
`firebase deploy`
